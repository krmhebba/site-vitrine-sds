# Site Vitrine SDS - Souidis

## Description
Site web institutionnel et catalogue en ligne développé pour la Société de Distribution Souibri - Souidis (SDS), distributeur d'outillage professionnel au Maroc. Ce projet permet à l'entreprise de présenter ses gammes de produits (Bellota, Urko), de recevoir des demandes de devis en ligne et de gérer le suivi des commandes via une interface connectée.

## Fonctionnalités
* Catalogue Produits : Présentation organisée par catégories (Agriculture, Construction, Atelier & Bricolage, Jardinage).
* Système de Devis en Ligne : Formulaire dynamique permettant aux clients de sélectionner des produits et des quantités.
* Gestion des Demandes (Backend) : Intégration avec Firebase pour stocker les demandes en temps réel.
* Interface d'Administration : Page sécurisée permettant de consulter les demandes, modifier leur statut (En attente, Traité, Devis envoyé) et les supprimer.
* Suivi Client : Possibilité pour les clients de consulter l'état de leurs demandes via leur adresse e-mail.
* Design Responsive : Adaptation de l'affichage pour mobiles et tablettes avec menu burger.
* Présentation Multimédia : Intégration de vidéo en arrière-plan sur la page d'accueil.

## Technologies Utilisées
* Frontend : HTML5, CSS3, JavaScript.
* Backend / Base de données : Google Firebase (Realtime Database).

## Structure du Projet
* index.html : Page d'accueil avec vidéo de présentation.
* Nos Produits.html : Catalogue détaillé avec images et descriptions.
* Demande de Devis.html : Formulaire de commande interactif.
* Liste des Demandes.html : Tableau de bord pour le suivi et la gestion administrative.
* style.css : Feuille de style globale et media queries.

## Installation
1. Cloner le dépôt.
2. Ouvrir le fichier index.html dans un navigateur web.
3. Une connexion internet est requise pour le fonctionnement du formulaire (connexion API Firebase).
